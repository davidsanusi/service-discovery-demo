API Explorer - Housing Data

This simple page is intended to demonstrate service discovery functionality with AWS ECS Service Discovery

Here, we have two services, frontend and backend. The frontend is this page you are currently viewing. The backend is hosted as another service and provides a set of housing data api endpoints that can be called using the form below

What to do: enter any of the available endpoints in the form and click on submit query

What will happen: This frontend service will make a call to the backend service to get the data provided by the endpoint. The data is displayed in the section below the submit button.

The ip address of the instance that serviced the request is also returned.

Available endpoints : owners, color, location, type
